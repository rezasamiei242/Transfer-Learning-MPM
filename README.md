# Transfer-Learning-MPM

Source codes accompanying the paper:

**Transfer learning for mineral prospectivity mapping in limited occurrence areas: Methodological improvement and a case study from the Kerman Cenozoic Magmatic Arc, Iran**

This repository contains the source codes for the transfer learning framework developed in this study.

- Python implementation of Support Vector Regression (SVR)
- Python implementation of Artificial Neural Network (ANN)
- R implementation of Gaussian Process Regression (GPR)

A sample dataset is included for demonstration purposes.

The original datasets are not publicly available due to research data restrictions but are available from the corresponding author upon reasonable request.
